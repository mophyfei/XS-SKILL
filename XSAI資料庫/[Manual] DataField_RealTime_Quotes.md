## 估計量
--- 

### 估計量
#### 估計量 – （報價欄位） <kbd>常用</kbd> <kbd>量能</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 估計量 |
| 欄位分類 | 常用、量能 |
| 代碼 | `EstimatedTotalVolume`<br>`q_EstimatedTotalVolume` |
| 單位 | 個股:張，大盤:元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 大盤、台(股票) |

##### 語法
> 當日收盤的預估成交量。也可以使用 GetField("估計量", "D") 來表示。  

```pascal
Value1 = GetQuote("估計量");
Value1 = GetQuote("EstimatedTotalVolume");
Value1 = q_EstimatedTotalVolume
```

---

##### 說明
當日收盤的預估成交量。也可以使用 GetField("估計量", "D") 來表示。

大盤或是個股的估計的方式是統計過去一段時間內每日盤中的分鐘累計成交量，算出每分鐘成交量的分佈比例，然後依照當日開盤迄今的累計成交量，來推算收盤時可能的成交量。

類股商品（細產業指標、主題指標）的計算方式為成分股預估量乘以均價的加總。

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 參考價 |
| 欄位分類 | 常用、價格 |
| 代碼 | `RefPrice`<br>`q_RefPrice` |
| 單位 | 元 |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台股 |

##### 語法
> 當日的參考價。也可以使用 GetField("RefPrice", "D") 來表示。  

```pascal
Value1 = GetQuote("參考價");
Value1 = GetQuote("RefPrice");
Value1 = q_RefPrice
```

---

##### 說明
當日的參考價。也可以使用 GetField("RefPrice", "D") 來表示。

一般而言是前一交易日的收盤價，如果股票商品有除權息的話，則是除權息後的參考價。

除權息的計算公式如下(實際數值以交易所公佈為主)：
1.除息參考價為股價－股息
2.除權參考價為股價／(1+配股率)
3.同時除權息參考價為 (股價-股息)／(1+配股率)

## 溢價率百分比

### 溢價率百分比

> 損益兩平價格與標的價格的差異幅度比例。數值越大，代表要達到損益兩平越不容易，風險就越大。

Value1 = GetQuote("委賣");

## 暫緩撮合狀態
--- 

### 暫緩撮合狀態
#### 暫緩撮合狀態 – （報價欄位） <kbd>其他</kbd>

| 項目 | 說明 |
| ---- | ---- |
| 欄位名稱 | 暫緩撮合狀態 |
| 欄位分類 | 其他 |
| 代碼 | `CircuitBreakState`<br>`q_CircuitBreakState` |
| 格式 | 數值 |
| 支援腳本 | 警示、交易、函數 |
| 支援商品 | 台(股票)、期貨 |

##### 語法
> **此欄位會回傳目前市場是否處於盤中價格穩定措施。**  

```pascal
Value1 = GetQuote("暫緩撮合狀態");
Value1 = GetQuote("CircuitBreakState");
Value1 = q_CircuitBreakState
```

---

##### 說明
回傳值代表的意義：
0 = 無
1 = 暫緩搓合且趨跌
2 = 暫緩搓合且趨漲

證交所價格穩定措施的流程:

如果盤中發生價格穩定措施時，此時q_MarketState = 0，而q_CircuitBreakState會是1 或 2。
接下來q_MarketState會變成8，此時可以抓到試搓價格。
等到價格穩定措施結束後，q_MarketState 和CircuitBreakState 會變回0，商品恢復正常交易。

如果是開盤時/收盤時發生價格穩定措施的話，則q_MarketState會是7 或 3。
